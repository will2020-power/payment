# payment

#### 介绍
支付系统学习，在线演示案例 [https://www.mixfate.com/pcpay/](https://www.mixfate.com/pcpay/)

#### 软件架构
vue + spring-boot 设计（学习中）

2.  部署后端

#### 使用说明

1.  打开模拟页
2.  选中微信支付，点击立即支付，打开二维码，等待扫码的过程中查询支付结果
3.  选中支付宝支付，点击立即支付，打开二维码，等待扫码的过程中查询支付结果
